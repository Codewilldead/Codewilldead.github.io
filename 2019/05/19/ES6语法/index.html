ES6语法

ES6 = ECMAScript 6

申明变量let和const

let

1、let申明变量不存在预解析；

2、let声明的变量不允许重复

3、ES6引入了块级作用域

块内部定义的变量，在外部无法访问(PS: 只要被 { } 包裹起来，就形成块级作用域)

const

const声明的常量不能被重新赋值，且在申明的同时就要被初始化

箭头函数的注意事项

1、箭头函数中this取决于函数的定义，而不是函数的调用

2、箭头参数不可以new

3、箭头函数不可以使用arguments获取参数列表，可以使用rest代替
